Aula 04 — Mochila Fracionária

Código Java

import java.util.Arrays;
import java.util.Comparator;

public class MochilaFracionaria {
    public static void main(String[] args) {
        int capacidade = 50;
        int[] pesos = {10, 20, 30};
        int[] valores = {60, 100, 120};
        int n = valores.length;

        double[][] itens = new double[n][3];
        for (int i = 0; i < n; i++) {
            itens[i][0] = pesos[i];
            itens[i][1] = valores[i];
            itens[i][2] = (double) valores[i] / pesos[i];
        }

        Arrays.sort(itens, Comparator.comparingDouble(o -> -o[2]));

        double valorMaximo = 0;
        int capacidadeRestante = capacidade;

        for (int i = 0; i < n; i++) {
            if (capacidadeRestante >= itens[i][0]) {
                valorMaximo += itens[i][1];
                capacidadeRestante -= itens[i][0];
            } else {
                valorMaximo += itens[i][1] * ((double) capacidadeRestante / itens[i][0]);
                break;
            }
        }

        System.out.println("Valor máximo obtido: " + valorMaximo);
    }
}

Análise

O algoritmo escolhe itens com base na maior razão valor/peso, preenchendo a mochila até atingir a capacidade.

Essa é uma solução gulosa e eficiente, pois sempre prioriza o melhor benefício por unidade de peso.

Complexidade: O(n log n), devido à ordenação inicial.

Resultado: Com capacidade 50, o valor máximo obtido é 240.0.

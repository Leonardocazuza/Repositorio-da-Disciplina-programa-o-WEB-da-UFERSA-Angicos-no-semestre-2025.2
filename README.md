Este repositório reúne um conjunto de atividades desenvolvidas ao longo da disciplina de Análise de Algoritmos, explorando conceitos fundamentais como eficiência computacional, estratégias gulosas, métodos de busca, estruturação de soluções e interpretação de desempenho.

Todo o conteúdo foi organizado e apresentado utilizando 100% HTML, permitindo visualizar de forma clara e estruturada cada etapa dos exercícios, incluindo códigos-fonte, explicações conceituais, tabelas comparativas, análises de complexidade e interpretações dos resultados obtidos.

As páginas construídas em HTML funcionam como mini-relatórios interativos, facilitando o estudo, a revisão do conteúdo e o acompanhamento da evolução das atividades ao longo da disciplina. Cada arquivo foi elaborado para fornecer uma compreensão acessível e didática das soluções propostas, mantendo a apresentação consistente e bem organizada.

Este repositório, portanto, não apenas documenta o desenvolvimento das tarefas da disciplina, mas também serve como material de consulta para revisões futuras, consolidando os aprendizados obtidos durante o estudo dos algoritmos e de sua aplicação prática.

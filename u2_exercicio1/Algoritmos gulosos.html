Aula 03 — Cálculo de Troco

Código Java

public class Troco {
    public static void main(String[] args) {
        int[] cedulasDisponiveis = {50, 20, 10, 5, 2, 1};
        int trocoPara = 93;
        int[] cedulasUsadas = new int[cedulasDisponiveis.length];

        for (int i = 0; i < cedulasDisponiveis.length; i++) {
            if (trocoPara >= cedulasDisponiveis[i]) {
                cedulasUsadas[i] = trocoPara / cedulasDisponiveis[i];
                trocoPara = trocoPara % cedulasDisponiveis[i];
            }
        }

        System.out.println("Cédulas usadas:");
        for (int i = 0; i < cedulasDisponiveis.length; i++) {
            if (cedulasUsadas[i] > 0) {
                System.out.println("R$ " + cedulasDisponiveis[i] + ",00 -> " + cedulasUsadas[i] + " cédula(s)");
            }
        }
    }
}

Análise

O código percorre as cédulas da maior para a menor, utilizando o método ganancioso (greedy), que sempre escolhe a maior cédula possível primeiro.

Essa abordagem garante o menor número total de cédulas e é eficiente para sistemas monetários comuns.

Complexidade: O algoritmo é O(n), pois percorre o vetor de cédulas apenas uma vez.

Conclusão: A solução é simples, rápida e garante um troco otimizado.
